# The bridge runs on the Python standard library alone — including the real
# OpenRouter client (plain urllib POST to the OpenAI-compatible Chat
# Completions endpoint). There is NO runtime third-party dependency: --stub
# mode, the test suite, and live play all run with nothing installed.
#
# Provider-agnostic by design (CLAUDE.md): the model is the YOMI_MODEL env var
# (any OpenRouter slug), the key is OPENROUTER_API_KEY (Infisical-injected),
# and the endpoint is OPENROUTER_BASE_URL — so swapping models/providers never
# touches code or this file.

# Dev/test dependency only (install manually if you want to run the suite):
